# podpod
24.12 ~ 25.07

## 프로젝트 설명
- 나의 성향에 맞는 K-POP 덕질 메이트를 찾아 덕친(덕질친구) 매칭 서비스

## 맡은 업무
- 백엔드 API 개발
- CI/CD
- 데이터 스크래핑 관리

## 사용된 기술
- Framework : FastAPI
- Language : Python 3.12
- DB : PostgreSQL
- CI/CD : fly.io(PaaS), Neon(DB-PostgreSQL)
- ETC : firebase storage(이미지 클라우드)

## 주요 개발 목록
- User 및 파티시스템(매칭 시스템 핵심 서비스)
- 파티 시스템 주소 처리
- 스크래핑된 데이터 가공(아티스트 스케쥴 관련)
- 파티 게시글 추천 알고리즘(가중치)
- 유저 성향 테스트 알고리즘(가중치)
